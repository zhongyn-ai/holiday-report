春节爆发系数远高于清明/五一，中低线旅游城市正餐系数1.72，本地地方菜高线1.94；量价齐升

一、整体爆发系数（品类×城市线级）

X轴=整体/高线/中低线旅游/中低线非旅游；额/量/价三柱并排，同时展示系数与YoY

2026年春节餐饮大盘在出游热潮的持续强化下，展现出更为显著的节假日虹吸效应。根据美团收银数据，高线 / 中低线非旅游 / 中低线旅游城市的日均流水爆发系数分别达 1.28 / 1.73 / 1.70（YoY +0.08 / +0.05 / +0.04），节假日消费势能进一步放大；其中单量爆发（1.03 / 1.38 / 1.35，YoY +0.05 / +0.03 / +0.03）为核心驱动，单均价则相对平稳（YoY +0.02 / 持平 / 持平）。伴随大盘出行客流的扩容，异地游客对"体验当地文化"的诉求集中爆发，使得本地地方菜/特色菜稳居最强势品类；同时，极具地域特质的细分赛道（华北/东北烧烤、云贵特色火锅、川渝辣锅）亦成功承接外溢流量，爆发力不输主流本地地方菜；东北赏雪/华南避寒分别作为春节传统/新晋热门，本地特色落在海鲜小馆/排档品类，均带动鱼鲜海鲜爆发增强。

从人口流动轨迹看，"先返乡、后出游"的多段式行程与"反向出游"结构性变化，共同塑造城市线级与品类的时序错位爆发。① 前半程（除夕-初三）：受返乡大潮驱动，中低线城市同比爆发更强；② 后半程（初三-初七）：受"反向出游"热潮驱动，人群轨迹向高线切换，高线城市爆发同比在初三追平中低线，并于初五至初七叠加"复工潮"实现反超。轻餐强势品类亦随出游节奏而交替：前半程面包甜点在各点位全面走强，后半程出游高峰期（初三-初六）奶咖街边店接力爆发。

三、轻餐渠道日度 YoY（商场店 vs 街边/社区店）

新中式茶饮/面包甜点/咖啡。暖色=正增长，冷蓝=负增长

暖冬叠加春节较晚的天气利好，进一步重塑了线下客流的物理分布与品类分化。天气转暖下，消费者更愿在户外活动而非封闭商场内停留，街边/社区点位凭借更高的可触达性和即时性流水爆发系数YoY显著走强，并对商场店形成挤压（轻餐整体爆发系数YoY 街边/社区店 +0.10 vs 商场店 -0.03）；品类层面，烧烤/烤串受天气利好表现强劲，而辣锅等偏暖身品类相对利空（爆发系数YoY 烤串 +0.11 vs 辣锅 -0.01）。值得注意的是，尽管受天气利好及千问短期拉动，茶饮头部品牌春节同店农历同比普遍仅录得高单位数增长；逆势领跑者如古茗（同店 YoY ~20%）实质是依靠基本面改善（咖啡门店渗透拉动同店~10%、新品成功）。
