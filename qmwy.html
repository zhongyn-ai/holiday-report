一、整体爆发系数（品类×城市线级）

X轴=整体/高线/中低旅游/中低非旅，清明+五一对比

整体爆发表现：五一受南部多地暴雨、清明春假连休分流双重压制，正餐爆发在高基数下同比下滑，并以中低线旅游城市尤甚。根据美团收银数据，26年五一高线 / 中低线非旅游 / 中低线旅游城市的正餐日均流水爆发系数分别 1.13 / 1.37 / 1.42（YoY 持平 / -0.01 / -0.04）；下滑由单量爆发收缩导致（YoY -0.02 / -0.02 / -0.04）、单均价则稳中有增（YoY +0.02 / +0.02 / 持平），反映在虹吸效应下商家"趁节日多赚"心态强化。清明虽有春假连休助益，但正餐日均流水爆发系数仅微幅回升（YoY +0.02 / +0.01 / 持平），且结构上由单均价而非单量拉动，爆发增长已趋疲软。轻餐则因25年五一外卖战遗留"量高价低"的异常基数，26年五一量价结构同比常态化回调，而单店流水口径同比基本持平。

二、正餐 品类×价格带（爆发系数YoY）

纵向=品类分组（含自助系），横向=价格带。暖色=同比正增长，冷色=负增长

正餐品类结构：地方菜/特色菜主导爆发，节假日品质消费亦向其收拢。本地地方菜/特色菜作为出游场景下"吃本地特色"的首要受益品类，五一依然保持最强爆发；清明则因春假拼假助益长途出行增长，本地特色品类同比增强趋势显著。消费K型分化下，"品类×价格带"的心智定位在加速清晰：① 地方菜/特色菜向高价格带集中，承接节假日出游场景下的品质消费需求，低价格带爆发相应减弱；② 火锅/自助收敛至"日常+平价"定位，节假日高价格带爆发同比大幅减弱；③ 烧烤在高价格带跑出细分机会，依托地域特色打造差异化/品质感，承接部分节假日"吃本地特色"消费需求。

三、品类×地域（五一爆发系数YoY）

每组品类单独出图，只展示YoY，颜色深浅区分品类，8大区对比

区域结构：南弱北强，长途游目的地受益明显、本地特色品类爆发同步走强。五一南部暴雨迫使客流北移，中南/西南在高基数下同比下滑；其余区域爆发同比改善，但客流构成决定了品类走势分化。26年五一长途游结构性提升（多地春假连休 + 出境减弱下客流回流国内），东北/西北作为长途游核心目的地直接受益，华北受小众目的地/文化游崛起吸引更多外地客流，三者均以外地客流为主，本地地方菜/特色菜爆发增强更为突出，"吃本地特色"倾向清晰可见；华东则因苏超"票根经济"+小众目的地带动爆发增强，但在长短途并行下，本地与异地地方菜同比趋势相近。
